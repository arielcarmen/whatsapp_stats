# whatsapp_stats

Statistics for Whatsapp groups

# How to install ?

- clone the repo 
- python -m venv venv
- source venv/bin/activate (linux or mac) or
- venv\Scripts\activate (windows)
- pip install -r requirements.txt

Then just wait.

After installation finished, to launch the app:
- streamlit run main.py
  
Enjoy !
